How to publish:
Upload the full robot-wholesaler-retail folder contents to your hosting provider. Keep the images folder beside index.html.

Before launch:
- Confirm the area spelling: Tuffontein or Turffontein.
- Add the real phone/WhatsApp number.
- Add real owner and team member photos to the Owner & Team section.
- Add business hours if available.
- Replace sample testimonials with real customer reviews when available.
- Replace generated images with real shop/warehouse photos if you receive the original uploaded images later.

How to add owner/team photos:
- Put the image files in the images folder.
- In index.html, replace each placeholder-photo block with an image tag, for example:
  <img class="team-photo" src="images/owner-photo.jpg" alt="Ramato Teshite Badane, owner of ROBOT WHOLESALER & RETAIL">
- Update "Team Member" names and roles before publishing.

Privacy note:
The website displays the owner name only. It does not publish any date of birth.
